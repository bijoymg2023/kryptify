⚙️ How It Works

1

Select Directory

Choose a folder to monitor. The scanner will recursively walk all files inside it.

2

Set Baseline

Capture the current state. SHA-256 hashes are computed for every file and stored as the trusted reference.

3

Run Scans

Re-hash all files and compare against the baseline. Any differences are flagged immediately.

4

Get Alerts

Added, modified, and deleted files are reported with colour-coded alerts and a full change history.
